gRPC-তে **method** মানে হলো **RPC (Remote Procedure Call)** — এক service অন্য service-এর একটি function দূর থেকে call করে।
এই method `.proto` ফাইলে define করা হয়।

তোমার example দিয়ে explain করি।

---

# 1️⃣ gRPC method কোথায় define হয়

`.proto` ফাইলে।

```proto
service AccountService {

  rpc GetAccount (AccountRequest) returns (AccountResponse);

}
```

এখানে:

```
rpc GetAccount (AccountRequest) returns (AccountResponse)
```

এইটাই **gRPC method**।

---

# 2️⃣ Method structure

Structure:

```
rpc MethodName (RequestMessage) returns (ResponseMessage)
```

Example:

```
rpc GetAccount (AccountRequest) returns (AccountResponse)
```

Meaning:

```
Client → AccountRequest
Server → AccountResponse
```

---

# 3️⃣ Java তে generated method

proto compile হলে Java class তৈরি হয়:

```
AccountServiceGrpc
```

এর ভিতরে method পাওয়া যাবে:

```java
public void getAccount(
        AccountRequest request,
        StreamObserver<AccountResponse> responseObserver)
```

---

# 4️⃣ Server side implementation

`account-service`

```java
@GrpcService
public class AccountGrpcService extends AccountServiceGrpc.AccountServiceImplBase {

    @Override
    public void getAccount(AccountRequest request,
                           StreamObserver<AccountResponse> responseObserver) {

        Long userId = request.getUserId();

        AccountResponse response =
                AccountResponse.newBuilder()
                        .setAccountId(1)
                        .setStatus("ACTIVE")
                        .build();

        responseObserver.onNext(response);
        responseObserver.onCompleted();
    }
}
```

এখানে:

| Part                        | Meaning          |
| --------------------------- | ---------------- |
| `request`                   | client request   |
| `responseObserver.onNext()` | response send    |
| `onCompleted()`             | request complete |

# 7️⃣ REST vs gRPC method

| REST                 | gRPC               |
| -------------------- | ------------------ |
| `GET /accounts/{id}` | `rpc GetAccount()` |
| JSON                 | Protobuf           |
| Controller method    | RPC method         |

Example REST:

```java
@GetMapping("/accounts/{id}")
```

Example gRPC:

```
rpc GetAccount (AccountRequest) returns (AccountResponse)
```

# 8️⃣ gRPC method types (important)

gRPC-তে **৪ ধরনের method** আছে।

### 1️⃣ Unary (সবচেয়ে common)

```
1 request → 1 response
```

```
rpc GetAccount(AccountRequest) returns (AccountResponse);
```

---

### 2️⃣ Server Streaming

```
1 request → many responses
```

Example:

```
rpc GetTransactions(TransactionRequest) returns (stream TransactionResponse);
```

---

### 3️⃣ Client Streaming

```
many request → 1 response
```

Example:

```
rpc UploadTransactions(stream TransactionRequest) returns (TransactionResponse);
```

---

### 4️⃣ Bidirectional Streaming

```
many request ↔ many response
```

Example:

```
rpc Chat(stream Message) returns (stream Message);
```
