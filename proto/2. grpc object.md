gRPC-তে “object” বলতে সাধারণত **Proto Message Object** বোঝায়।
এগুলো `.proto` ফাইলে `message` দিয়ে define করা হয় এবং compile হলে **Java classes** তৈরি হয়। এই objects ব্যবহার করে **client ↔ server data exchange করে**।

তোমার example দিয়ে পরিষ্কার করে ব্যাখ্যা করি।

---

# 1️⃣ gRPC Object কোথা থেকে আসে

`.proto` ফাইলে `message` লিখলে সেটা object হয়ে যায়।

```proto
message AccountRequest {
  int64 userId = 1;
}

message AccountResponse {
  int64 accountId = 1;
  string status = 2;
}
```

এখানে:

| Message           | Object type     |
| ----------------- | --------------- |
| `AccountRequest`  | request object  |
| `AccountResponse` | response object |

---

# 2️⃣ Compile হলে Java Object তৈরি হয়

Proto compile হলে Java classes তৈরি হয়:

```
AccountRequest.java
AccountResponse.java
```

Package:

```
snvn.grpc
```

---

# 3️⃣ Object structure (Java)

Generated class internally immutable object হয়।

Example (conceptually):

```java
public final class AccountResponse {

   private long accountId;
   private String status;

}
```

কিন্তু gRPC object **direct constructor ব্যবহার করে না**।

---

# 4️⃣ Builder pattern ব্যবহার হয়

gRPC objects তৈরি করতে **builder pattern** ব্যবহার করা হয়।

Example:

```java
AccountResponse response =
        AccountResponse.newBuilder()
                .setAccountId(1)
                .setStatus("ACTIVE")
                .build();
```

Step:

```
newBuilder()
   ↓
set field
   ↓
build()
```

---

# 5️⃣ Request object

Client request তৈরি করে:

```java
AccountRequest request =
        AccountRequest.newBuilder()
                .setUserId(10)
                .build();
```

---

# 6️⃣ Object access করা

Response পাওয়ার পরে:

```java
long accountId = response.getAccountId();
String status = response.getStatus();
```

---

# 7️⃣ gRPC object vs normal DTO

| Feature       | gRPC Object | DTO             |
| ------------- | ----------- | --------------- |
| Source        | `.proto`    | Java class      |
| Serialization | protobuf    | JSON            |
| Mutable       | ❌ immutable | usually mutable |
| Creation      | builder     | constructor     |

---

# 8️⃣ Data flow

```
Client
   │
   │ AccountRequest object
   ▼
Server
   │
   │ AccountResponse object
   ▼
Client
```

---

# 9️⃣ Serialization

gRPC object automatically convert হয়:

```
Java Object
    ↓
Protobuf Binary
    ↓
Network
    ↓
Java Object
```

এটা JSON এর চেয়ে অনেক দ্রুত।
