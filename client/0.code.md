# 🔥 What this class actually is?

```java
@Service
public class AccountGrpcClient
```

👉 This is a **gRPC client wrapper service**

✔ Used inside your **User Service**
✔ Calls **Account Service (remote microservice)**

---

# ⚡ Key Part 1: `@GrpcClient`

```java
@GrpcClient("account-service")
private AccountServiceGrpc.AccountServiceBlockingStub accountStub;
```

### 💡 What happens here?

* `"account-service"` must match your config:

```yaml
grpc:
  client:
    account-service:
      address: static://localhost:9093
      negotiationType: plaintext
```

👉 Spring automatically:

1. Connects to `localhost:9093`
2. Creates a **stub (proxy client)**
3. Injects it into `accountStub`

---

# ⚡ Key Part 2: Stub (`BlockingStub`)

```java
AccountServiceGrpc.AccountServiceBlockingStub
```

👉 This is generated from your `.proto` file

### 💡 Types of stubs:

| Stub Type      | Behavior                  |
| -------------- | ------------------------- |
| `BlockingStub` | Sync (waits for response) |
| `FutureStub`   | Async (future-based)      |
| `Stub`         | Stream-based              |

👉 You are using:
✔ **Blocking (simple + good for beginners)**

---

# ⚡ Key Part 3: Request Build

```java
AccountRequest request =
        AccountRequest.newBuilder()
                .setUserId(userId)
                .build();
```

👉 This is **proto object creation**

### 💡 Behind the scene:

Your `.proto`:

```proto
message AccountRequest {
  int64 userId = 1;
}
```

👉 Java generates:

* `newBuilder()`
* `setUserId()`
* `build()`

---

# ⚡ Key Part 4: Remote Call

```java
return accountStub.getAccount(request);
```

👉 This line does:

1. Sends request to:

```text
localhost:9093 (Account Service)
```

2. Calls method:

```proto
rpc GetAccount (AccountRequest) returns (AccountResponse);
```

3. Waits for response

4. Returns:

```java
AccountResponse
```

# 🧠 How you will use this in your code?

## Example:

```java
@RestController
public class UserController {

    private final AccountGrpcClient accountGrpcClient;

    public UserController(AccountGrpcClient accountGrpcClient) {
        this.accountGrpcClient = accountGrpcClient;
    }

    @GetMapping("/user/account/{id}")
    public AccountResponse getAccount(@PathVariable Long id) {
        return accountGrpcClient.getAccount(id);
    }
}
```

👉 Flow:

```text
Browser → REST → UserService → gRPC → AccountService
```
